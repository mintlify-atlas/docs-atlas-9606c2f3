## What is NextAudit AI?

NextAudit AI is built on a cyber-industrial microservices architecture that brings together best-in-class open source tools under a unified audit layer:

- **FleetDM** — Endpoint inventory, security posture, and osquery-powered visibility
- **n8n** — Visual workflow automation for audit pipelines and policy enforcement
- **Flowise** — AI agent flows for intelligent analysis and assisted response
- **Ollama** — Self-hosted LLM inference for private, on-premises AI
- **PostgreSQL + pgvector** — Operational data and AI embedding storage
- **MySQL + Redis** — FleetDM's persistence and caching layer

## Get started

<Steps>
  <Step title="Clone the repository">
    Get the source code and navigate to the project directory.
  </Step>
  <Step title="Configure environment variables">
    Copy the environment template for your target environment (`dev`, `test`, or `prod`) and fill in your credentials.
  </Step>
  <Step title="Launch the stack">
    Use Docker Compose to bring up all services — fleet, AI, and databases — in a single command.
  </Step>
  <Step title="Access the dashboard">
    Open FleetDM in your browser to start enrolling endpoints and configuring audit policies.
  </Step>
</Steps>
